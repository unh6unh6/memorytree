# Memory Tree 개발 순서

## Phase 1. 프로젝트 기반 설정

- [x] 폴더 구조 설계 (`src/styles/`, `src/scripts/`, `src/components/`)
- [x] 공통 CSS 변수 작성 (색상, 타이포그래피, spacing, elevation)
- [x] Inter 폰트 및 CSS 리셋 적용
- [x] Hash-based SPA 라우터 구현 (`#explorer`, `#list`, `#test`)

## Phase 2. 데이터 레이어

- [x] 핵심 데이터 모델 정의
  - `TreeNode { id, parentId, name, type: 'category' | 'qa', children[] }`
  - `QANode { id, question, answer, importance(0-5), createdAt }`
  - `Attempt { id, nodeId, result: 1.0 | 0.5 | 0.0, timestamp }`
- [x] LocalStorage CRUD 서비스 (`nodeService`, `attemptService`)
- [x] 초기 샘플 데이터 (CS > Data Structures, Algorithms 예시 포함)
- [x] 앱 시작 시 root 노드 초기화 보장
  - `parentId: null, name: 'My Tree', type: 'category'`인 노드가 없으면 자동 생성
  - seedData와 분리된 독립 로직 (`initRootNode`)으로 구현
  - root 노드 ID를 고정 상수로 관리 (`ROOT_NODE_ID = 'root'`)

## Phase 3. 레이아웃 & 공통 컴포넌트

- [x] 토스트 알림 컴포넌트

## Phase 4. 탐색기 화면 (Explorer / Canvas)

- [x] 75% 캔버스 영역 + 방사형 그리드 배경 렌더링
- [x] SVG 기반 트리 시각화 (원형 노드 + curved path 연결선)
- [x] 노드 클릭 시 플로팅 상세 패널 (이름, 타입, 메모리 강도, Edit/View 버튼)
- [x] 노드 추가/편집/삭제 (카테고리 & Q&A, 형제 노드 타입 제약 적용)
- [x] 줌 인/아웃/센터 버튼
- [x] 노드 좌클릭: 선택만 (플로팅 패널 미표시) / 우클릭: 플로팅 상세 패널 표시
- [x] 캔버스↔사이드바 선택 양방향 동기화
  - 사이드바에서 노드 선택 시 캔버스 노드도 선택 상태로 전환
  - 캔버스에서 노드 선택 시 사이드바 노드도 선택 상태로 전환
- [x] 사이드바 노드 우클릭 → 컨텍스트 패널 (클릭 위치 기준, 편집/삭제/모아보기 기능 제공)
- [x] 캔버스 노드 드래그로 자유 위치 이동 (레이아웃 수동 오버라이드)
- [x] QA 노드에 자식 추가 불가 — 리프노드 제약 UI 적용 (+ 버튼 비표시)
- [x] 사이드바 너비 드래그 조절 (우측 경계면 리사이저 핸들)

## Phase 4-2. 탐색기 화면 추가 기능

- [x] 상단바 Memory Tree 로고 클릭 시 탐색기 화면으로 즉시 이동
- [x] 노드 검색창을 상단바에서 좌측 디렉터리 상단으로 이동 (탐색기 전용 기능)
- [x] 카테고리 노드 항상 접기/펼치기 토글 표시 (자식 노드 없어도 표시)
- [x] 좌측 디렉터리 인라인 노드 추가 (카테고리 및 Q&A 추가 버튼)
- [x] 좌측 디렉터리 Shift 다중선택 + 우클릭 시 일괄 삭제만 제공
- [x] 캔버스 Shift 다중선택 + 우클릭 시 일괄 삭제만 제공
- [x] 좌측 디렉터리 드래그로 노드 이동
  - 드래그 중 대상 노드 하이라이트 표시
  - 다중선택 상태에서 드래그 시 선택된 모든 노드 일괄 이동
  - 이동 불가 조건 적용 (QA 노드 하위 이동 불가, 카테고리와 QA 혼합 불가 등)

## Phase 4-3. 탐색기 화면 추가 기능 3

- [x] 캔버스 러버밴드 선택: 빈 영역에서 좌클릭 드래그로 사각 영역을 그려 포함되는 노드들을 다중선택
- [x] 사이드바↔캔버스 다중선택 양방향 동기화
  - 사이드바에서 다중선택 시 캔버스에서도 해당 노드들이 다중선택 상태로 표시
  - 캔버스에서 다중선택 시 사이드바에서도 해당 노드들이 다중선택 상태로 표시 (조상 펼침 포함)
- [x] 캔버스 다중선택 노드 함께 이동 (그룹 드래그)
- [x] 노드 검색 시 매칭되는 노드가 표시되는 깊이까지 디렉터리 자동 펼침
- [x] 캔버스 우측 하단 확대/축소 버튼 제거
- [x] 원점 복귀 버튼 클릭 시 애니메이션으로 부드럽게 이동
- [x] 캔버스 노드 재배치 버튼 (원점 복귀 버튼 상단)
  - 꼬인 간선이 없고 일정한 간격으로 보기 좋게 노드/간선 자동 재배치
  - 선택 또는 다중선택된 노드가 있을 시 해당 노드와 하위 노드들만 재배치
  - 재배치 시 노드들이 애니메이션으로 부드럽게 이동 (600ms easeInOut)
